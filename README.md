# Automation Exercise Selenium Project

This project contains automated test cases for the website:
https://automationexercise.com  
using Python and Selenium.

What this project does
- Automated all 26 test cases from Automation Exercise
- User Signup and Login
- Add to Cart and Checkout
- Payment and Order Confirmation
- Invoice Download
- Account Deletion
- Address Verification and Assertions

Tech Stack
- Python
- Selenium
- Chrome WebDriver
